# 功能

* server端：
  * 充当管理员的角色
  * 监听所有client端的消息
  * 可以将指定的client下线
  * 可以以广播的方式发送消息给所有的client
  * 可以发送给指定的用户

* client端：
  * 支持多client端
  * 支持client起名字
  * 支持与特定的client聊天
